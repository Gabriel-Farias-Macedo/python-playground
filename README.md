# Analift

Analift é uma aplicação em Python voltada para análise de movimentos em exercícios de powerlifting.  
O projeto integra conceitos de treinamento sistematizado de Boris Sheiko, com foco em verificar se a execução dos levantamentos mantém o padrão técnico estabelecido.  

## Objetivo
O sistema busca otimizar métricas e padronizar movimentos, fornecendo feedback sobre consistência técnica e alinhamento com parâmetros de referência.  

## Funcionalidades previstas
- Captura de vídeo ou integração com sensores de movimento.  
- Extração de pontos-chave do corpo (pose estimation) com OpenCV/MediaPipe.  
- Comparação da execução com padrões técnicos baseados em metodologias de Sheiko.  
- Relatórios quantitativos sobre amplitude, velocidade, simetria e alinhamento articular.  
- Evolução futura para integração com modelos de machine learning, permitindo recomendações personalizadas.
